# open-voice-mode

Simple iPhone shortcuts that open AI voice mode with a Siri trigger — e.g. "Hey Siri, talk to Claude". Currently covers Claude, ChatGPT, and Gemini.

- "Hey Siri, talk to Claude" → opens Claude app in voice mode
- "Hey Siri, talk to ChatGPT" → opens ChatGPT app in voice mode
- "Hey Siri, talk to Gemini" → opens Gemini app in voice mode

## Install

- Claude: ~30 seconds
- ChatGPT: ~30 seconds
- Gemini: ~30 seconds

### ChatGPT

**Prerequisites:** Native Apple Shortcuts app & ChatGPT iOS app

**Quick Install:** 

On your iPhone, click on the link below and tap **Add Shortcut** in the preview. The link will create a new shortcut through the native Apple Shortcuts app.

https://www.icloud.com/shortcuts/a243d6a0ef5148b181a7fa79265c7dae

**Manual Install:** 

 - Open Shortcuts app
 - Tap **+** (top right) to create a new shortcut
 - Tap **Add Action**
 - Scroll the app list and tap **ChatGPT**
 - Tap **Start a Voice Conversation**
 - Rename the shortcut to **Talk to ChatGPT** (this is the Siri trigger phrase)

**Test:** say "Hey Siri, talk to ChatGPT". The ChatGPT app should open directly in voice mode.
